## Security

This project is open source and developed on a voluntary basis.
There is no financial expectation, paid support, or guaranteed response time.

The project has not undergone a formal security audit.

Please **do not report security vulnerabilities via public issues or pull requests**.

If you discover a potential security issue, contact the developer directly with:
- A clear description of the issue
- Reproduction steps or proof of concept, if available
- Affected versions or commits
